A capability that exposes a Microsoft Power Automate / Power Platform flow as a single MCP tool with shaped output, so a Copilot agent can call a curated org flow without re-implementing it. Power Platform is the M365 Copilot extensibility surface most ISV partners ship into; making flows MCP-callable is the bridge.
