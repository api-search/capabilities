# API Search - Capabilities

This repository stores workflow and capability definitions for the [APIs.io](https://apis.io) search network.

## About

Capabilities describe what APIs can do at a functional level, mapping API operations to business workflows. They are generated from api-evangelist repos and used to build the `_capabilities/` Jekyll collection on the APIs.io network site.

## Related Repositories

- [apis](https://github.com/api-search/apis) - Individual API records
- [providers](https://github.com/api-search/providers) - API provider records
- [schemas](https://github.com/api-search/schemas) - JSON Schema references
- [network](https://github.com/api-search/network) - The APIs.io network site
